HOW IT WORKS
Managers submit an Airtable Form at 1PM, 4PM, and 7PM each day.
Data lands in your Airtable base instantly.
Hit LOAD LIVE DATA below — this dashboard reads it and updates every KPI, every store, every call script.

STEP 1 — YOUR AIRTABLE CREDENTIALS
Go to airtable.com/account → Developer Hub → Personal access tokens → Create token (scope: data.records:read)
STEP 2 — AIRTABLE TABLE FIELDS
Create a table called Submissions with these fields:
Store — Single line text
CheckIn — Single select (1PM Check-In / 4PM Check-In / 7PM Close)
Date — Date
Voice, BTS, HSI, Reacts, Upgrades, ACC, Magenta — Number
Boxes — Formula: {Voice}+{BTS}+{Upgrades}
STEP 3 — MANAGER FORM
In Airtable → your Submissions table → click Forms → Create a form → name it Edge Link — Daily Check-In
Add all fields. Set Store as a dropdown with all 34 store names. Set CheckIn with three options: 1PM Check-In, 4PM Check-In, 7PM Close.
Click Share form → copy the link → send to all 34 managers. They fill it out at 1PM, 4PM, 7PM. Done in 60 seconds.
