SETUP — CONNECT YOUR DATA
STEP 1 — UPLOAD EXCEL TO SHAREPOINT
Upload EdgeLink_April_Command_Center.xlsx to your SharePoint/OneDrive. Make sure the ACTUALS sheet exists (Power Automate writes form submissions here).
STEP 2 — CREATE MICROSOFT FORM
Go to forms.microsoft.com → New Form → name it Edge Link — Daily Store Report

Add these fields in order:
1. Store Name (Dropdown — all 34 stores)
2. Check-In Time (Choice: 1PM Check-In / 4PM Check-In / 7PM Close)
3. New Voice (Number)
4. BTS Tablets (Number)
5. HSI / HINT (Number)
6. Reacts (Number)
7. Upgrades (Number)
8. Accessory Revenue $ (Number)
9. Magenta in Metro (Number)

Settings → One response per person: OFF → Share link with all managers.
STEP 3 — POWER AUTOMATE FLOW
Go to make.powerautomate.com → New Flow → Automated

Trigger: When a new response is submitted (Microsoft Forms)
Action 1: Get response details (Microsoft Forms)
Action 2: Add a row into a table (Excel Online — Business)
   • Location: SharePoint  • File: your Excel  • Table: ACTUALS
   • Map: Store=Store Name, CheckIn=Check-In Time, Voice=New Voice, BTS=BTS Tablets, HSI=HSI/HINT, Reacts=Reacts, Upgrades=Upgrades, ACC=Accessory Revenue, Magenta=Magenta in Metro

Save and test with a sample submission.

In SharePoint: right-click your Excel file → Share → Copy link
Change /embed? to /download? in the URL (or add &download=1 to the end)
